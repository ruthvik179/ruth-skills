# REPORT.md Fields

`REPORT.md` must include:

- Job URL, company, title, location when available.
- Scrape status, fallback requirement, and extraction message.
- Selected summary.
- Selected skills and keyword rationale.
- Final bullets, source IDs, and rationale for each rewrite.
- Gap notes for unsupported important requirements.
- ATS checklist.
- Render results including page count and PDF text extraction status.

Gap notes must record missing requirements instead of inventing resume claims.
